# SuperApp на Python — Day 11

Модульное десктопное приложение, объединяющее несколько независимых утилит под единым графическим интерфейсом.

##  Обоснование стека (UI & Graphics)

Для реализации интерфейса и интерактивных графиков выбрана связка **Tkinter (Ttk)** + **Matplotlib**:
1. Tkinter / Ttk: Встроенная библиотека Python, не требующая тяжелых внешних зависимостей, обеспечивающая высокую скорость работы и нативную кроссплатформенность. Использование кастомной стилизации через `ttk.Style` позволяет добиться современного плоского (Flat) дизайна UI.
2. Matplotlib (Backend TkAgg): Промышленный стандарт для работы с графиками в Python. Он предоставляет полноценную интерактивность (масштабирование, перемещение, сохранение снимков) и бесшовно встраивается в контейнеры (Frames) Tkinter.

##  Архитектурные принципы
- Low Coupling (Слабая связанность): Главное окно ничего не знает о внутренней логике модулей. Оно лишь предоставляет контейнер для отрисовки.
- Open-Closed Principle: Приложение открыто для расширения (добавления новых утилит в `ui/modules/`), но закрыто для изменения основного ядра навигации.
